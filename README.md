# Cart Functionality in React

This project demonstrates a simple **Add to Cart** functionality using React. It includes features like:

- Displaying products
- Adding items to the cart
- Quantity management
- Total item count and cost display

## Tech Stack

- React
- JavaScript
- useState Hook
- Basic CSS
